## Instructions:

### Prerequisites

* Install Maven (http://maven.apache.org)
* Alternatively use IDE and run the app main() method from the Application class

### Run:

With Maven installed run following commands in the project route folder:  

1. `mvn compile'

2. `mvn exec:java -Dexec.mainClass="Application"`

### Run tests:

Run the following command in the project route folder:

* `mvn clean test`

### Built with:

* Maven - Dependency Management
* JUnit 5 - Testing framework
